# Steward Styling Guide

## Colors
```tsx
// Badge
<div className="inline-flex items-center gap-2 px-4 py-2 bg-green-100 border-2 border-green-700 text-green-800 font-bold text-sm">
  <Leaf className="w-4 h-4" />
  PROTECT WHAT WORKS
</div>

// Stats
<div className="text-4xl font-black text-green-700">{count}</div>

// CTA
<Link className="bg-green-700 text-white border-2 border-black hover:bg-green-800">
  BECOME A STEWARD
</Link>

// Background
<section className="bg-green-50">
```

## Language

| Concept | Use This |
|---------|----------|
| Membership | "Steward" (not advocate, member) |
| Action | "Protect what works" |
| Community | "Nurture communities" |
| Data | "Protect knowledge" |
| Resources | "Guide resources" |

## Stewardship Principles

1. **Nurture, Don't Extract** - Grow community capacity
2. **Protect What Works** - Evidence-based programs deserve protection
3. **Community First** - Communities own their data
4. **Long-Term Thinking** - Steward for generations
