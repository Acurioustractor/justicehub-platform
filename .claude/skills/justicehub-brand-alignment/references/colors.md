# Color Palette

## Homepage/Marketing (White BG)
```css
--background: #FFFFFF
--foreground: #000000
--accent: hsl(0, 78%, 44%)     /* Bold red for urgent CTAs */
--success: #1e40af              /* Blue - community programs */
--failure: #ea580c              /* Orange - detention fails */
```

## ALMA Intelligence (Dark BG)
```css
--background: #0a0f16
--foreground: #FFFFFF
--alma-green: #27ae60           /* Success, positive */
--alma-orange: #e57a28          /* Warning, failures */
```

## Steward/Community (Green Accent)
```css
--steward-green: #15803d        /* green-700 */
--steward-green-light: #dcfce7  /* green-100 */
```

## Data Visualization Rules
- Green/Blue = Success, community programs work
- Orange/Red = Failure, detention system fails
- Always show comparison (success vs failure side by side)
