# Extraction Patterns

## Government Sites

```
Extract from this Australian government youth justice webpage:
- Program names and official titles
- Eligibility criteria
- Service locations
- Contact information
- Related legislation
- Outcome data if published

Focus on: Concrete program details, not general policy statements.
```

## Indigenous Organizations

```
Extract from this Indigenous organization webpage with cultural respect:
- Community-led programs and initiatives
- Cultural elements and approaches
- Self-determination principles
- Community outcomes (as they define them)

IMPORTANT: This is Community Controlled knowledge. Mark as such.
Do not extract without honoring sovereignty.
```

## Research Publications

```
Extract from this research publication:
- Study methodology
- Sample size and demographics
- Key findings with statistical significance
- Limitations acknowledged
- Policy implications
- Citations to other relevant research
```

## Media Articles

```
Extract from this media article about youth justice:
- Main narrative and framing
- Stakeholders quoted
- Statistics cited
- Policy positions mentioned
- Sentiment (supportive/critical/neutral)

Categorize: Is this about reform, punishment, or incident reporting?
```
