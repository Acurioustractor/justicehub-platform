# CONTAINED — "Nominated" Tile Prompts (ChatGPT / image generators)

The live system tile is generated automatically at:

```
https://justicehub.com.au/api/contained/nomination-tile?name=NAME&title=TITLE&reason=REASON
```

(URL-encode the values. Square 1080x1080 PNG, brand-locked.)

Use the prompts below only for bespoke graphic variants (illustration/poster style is approved for AI generation; never photorealistic people or places).

## Base prompt (paste into ChatGPT image generation)

> Design a square 1080x1080 social media tile for an Australian justice campaign called CONTAINED. Strict palette: near-black background #0A0A0A, off-white text #F5F0E8 (never pure white), urgent red #DC2626 used only for the stamp and one accent rule. Typography: bold condensed grotesque (like Space Grotesk Bold) for the name, monospace (like IBM Plex Mono) for labels. Layout: thin red bar top-left; small letterspaced uppercase kicker "CONTAINED · PUBLIC NOMINATION"; a large red rubber-stamp style box rotated slightly, reading "NOMINATED"; below it the name "[NOMINEE NAME]" in huge uppercase off-white, with "[TITLE]" smaller beneath; then a red left-rule quote block headed "WHY THEY NEED TO WALK THROUGH" containing: "[REASON, max 2 sentences]". Footer: "Three rooms. Ten minutes in each." / "Walk through. Then decide." in red, and "justicehub.com.au/contained" right-aligned. Background: a real photograph of the CONTAINED cell room (steel-framed bed, blue mattress, graffiti walls, container doors) darkened to ~85% black so the type carries. Edgy, stark, editorial protest poster energy. No photos, no faces, no AI-photorealism, no gradients, no extra colours.
