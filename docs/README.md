## Project Overview

JusticeHub is a Next.js 14 platform connecting system-impacted youth with support services, built with:
- Next.js 14.2.30 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (PostgreSQL)
- Vercel deployment
