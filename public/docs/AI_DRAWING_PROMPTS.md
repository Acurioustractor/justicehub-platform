## ðŸŽ¯ Platform-Specific Instructions

### For Miro AI:
1. Paste prompt into Miro AI chat
2. Request "professional infographic style"
3. Ask for "editable shapes" so you can adjust afterwards
4. Export as SVG for scalability

### For Napkin AI:
1. Paste prompt into Napkin input
2. Select "Infographic" style
3. Choose "Professional/Corporate" theme
4. Export as high-res PNG (minimum 1920x1080)

### For Canva AI (Magic Design):
1. Create new design (Presentation or Infographic size)
2. Use "Magic Write" with prompt
3. Request "professional corporate style"
4. Fine-tune colors and text manually
5. Export as PDF for print quality

### For Adobe Firefly:
1. Use "Text to Image" feature
2. Add to prompt: "professional infographic, vector style, corporate presentation, high contrast, clean design"
3. Generate multiple variations
4. Export highest resolution

### For Midjourney:
Add to each prompt:
```
--style professional infographic, corporate design, clean lines, high contrast, vector style, suitable for business presentation, white background --ar 16:9 --quality 2
```

### For DALL-E 3 (ChatGPT):
Add to each prompt:
```
Style: Professional corporate infographic suitable for McKinsey or Deloitte presentations. Clean, modern, high contrast. Vector-style illustration. White or light gray background. Print-ready quality. No photographs, pure graphic design.
```

## ðŸ“¥ How to Get Best Results

### Color Codes (use exact hex values):
- **Problem Red:** #DC2626
- **Solution Green:** #059669
- **Platform Blue:** #2563EB
- **Warning Orange:** #EA580C
- **Local Yellow:** #F59E0B
- **Policy Purple:** #7C3AED

### Typography:
- Request: "Modern sans-serif fonts like Inter, Roboto, or Helvetica"
- Hierarchy: Large titles, medium subtitles, small body text
- High contrast for readability

### Export Settings:
- **For presentations:** PNG at 1920x1080 or higher
- **For print:** PDF with minimum 300 DPI
- **For web:** SVG (vector, infinitely scalable)
- **For social:** Square 1500x1500 or Instagram 1080x1350

### Iteration Tips:
1. Generate first version
2. Request adjustments: "make text larger", "increase contrast", "adjust spacing"
3. Fine-tune colors to match exact hex codes
4. Test readability from distance (presentations)
5. Export multiple formats
