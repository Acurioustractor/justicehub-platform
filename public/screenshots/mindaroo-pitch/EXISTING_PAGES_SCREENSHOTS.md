## üí° TIPS

### For Better Screenshots:
1. **Clear cache** before capturing (fresh, clean state)
2. **Use sample data** that looks professional
3. **Remove test data** or placeholder text
4. **Ensure UI is complete** (no missing images/broken links)
5. **Check mobile responsive** if needed (375px width)
6. **Capture in light mode** (better for printing/presentations)

### For Admin Pages:
1. **Log in first** to access admin routes
2. **Show actual content**, not empty states
3. **Blur sensitive data** if any exists
4. **Highlight key features** with annotations if needed

### For Wiki Pages:
1. **Wait for MDX to render** fully
2. **Ensure visualizations load** (the 5 SVGs should be visible)
3. **Capture with sidebar** showing navigation
4. **Include scroll context** if page is long
