## üì∏ QUICK CAPTURE GUIDE

### Setup
```bash
# 1. Start dev server
PORT=4000 npm run dev

# 2. Open browser in incognito mode
# Chrome: Cmd+Shift+N (Mac) or Ctrl+Shift+N (Windows)
# Firefox: Cmd+Shift+P (Mac) or Ctrl+Shift+P (Windows)
```

### Capture Process
1. Visit URL in browser
2. Wait for page to fully load (no spinners)
3. Take screenshot (Cmd+Shift+3 on Mac, Win+Shift+S on Windows)
4. Save with exact filename from list above
5. Move to `/public/screenshots/mindaroo-pitch/platform/`
6. ‚úÖ Check off in this list

### For Full-Page Screenshots
Use browser extension like:
- **GoFullPage** (Chrome/Firefox)
- **Awesome Screenshot** (Chrome/Firefox)
- **Fireshot** (Chrome/Firefox)
